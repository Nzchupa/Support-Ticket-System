# 🎫 Support Ticket System

## 📌 Beschreibung

Dieses Projekt ist ein **Support Ticket System**, das zur Verwaltung von Support-Anfragen, Problemen und Aufgaben dient.
Solche Systeme werden häufig im IT-Support oder Kundenservice verwendet, um Anfragen strukturiert zu erfassen und zu bearbeiten. ([Wikipedia][1])

Ziel dieses Projekts:

* Tickets erstellen und verwalten
* Probleme strukturiert bearbeiten
* Teamarbeit und Workflows lernen
* Praxis mit Java, Git und GitHub sammeln

---

# 🚀 Features (Beispiele)

* 📝 Ticket erstellen
* 📂 Tickets anzeigen / filtern
* 🔄 Status ändern (z. B. Open, In Progress, Closed)
* 👤 Benutzer / Rollen (optional)
* 🔍 Suche nach Tickets
* 📊 Übersicht aller Anfragen

# ⚙️ Installation & Setup

## 1. Projekt klonen

```bash
git clone https://github.com/Nzchupa/Support-Ticket-System.git
cd Support-Ticket-System
```

## 2. Projekt öffnen

In Eclipse:

```
File → Import → Existing Project
```

Falls Fehler:

```
Right Click → Configure → Convert to Java Project
```

---

# ▶️ Projekt starten

* Hauptklasse ausführen (`Main.java`)
* oder über Eclipse „Run“
